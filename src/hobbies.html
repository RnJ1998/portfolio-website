OPC UA Server and Client Learning Project

  • Developed a demonstration of an OPC UA (Open Platform Communications Unified Architecture) server and client using Python.
  • Simulates a basic industrial process, exposing variables like "Temperature" and "Pressure" and a "Multiply" remote method.
  • The client connects to the server, browses its address space, reads/writes data, invokes methods, and subscribes to real-time updates.
  • Implemented secure communication using self-signed X.509 certificates for both server and client.
  • Containerized both applications using Docker and Docker Compose for easy setup and deployment.
